# 🏦 Loan Approval Prediction System

A Machine Learning web application that predicts whether a loan application is likely to be approved based on applicant details. The model is trained using the Random Forest Classifier and deployed with Streamlit.

---

## 📌 Project Overview

The goal of this project is to automate the loan approval prediction process by analyzing applicant information such as income, loan amount, credit history, property area, and other factors.

---

## 🚀 Features

- Predicts Loan Approval (Approved / Rejected)
- Interactive Streamlit Web Application
- Data Preprocessing and Feature Engineering
- Random Forest Machine Learning Model
- User-Friendly Interface
- Real-Time Prediction

---

- Source: Kaggle
